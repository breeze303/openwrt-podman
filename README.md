# openwrt-podman

源码来源：https://github.com/oskarirauta/luci-podman.git


使用方法：
```
git clone --depth 1 --single-branch https://github.com/breeze303/openwrt-podman package/podman
./scripts/feeds install -a
make menuconfig
```
